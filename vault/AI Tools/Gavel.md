# Gavel

Gavel (formerly Documate) provides no-code document automation for legal professionals. Attorneys can create interactive questionnaires that guide users through a series of questions, then automatically generate customized legal documents based on the responses. The platform is used by legal aid organizations, courts, and law firms to make legal document creation accessible and efficient.

## Features
- No-code interactive questionnaire builder
- Conditional logic for complex document generation
- Client-facing web forms with branded experience
- Multi-document generation from single intake
- API access for embedding automation in other platforms

## Best For
- Legal aid organizations creating self-service document tools
- Courts and government agencies automating form generation

## Pricing
Free tier for basic automation. Pro plan at $79/month. Business plan at $199/month with advanced features and API access. Enterprise pricing available.
